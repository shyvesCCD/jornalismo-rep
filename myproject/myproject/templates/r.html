Linguagem de programação R.

Para que serve o R?

R é uma linguagem de programação multi-paradigma, ou seja, podemos utilizar princípios tanto do paradigma funcional, sequencial e o orientado a objetos.
Porém o R brilha mesmo é na parte estátistica, com pacotes que facilitam muito na hora de desenvolver. Aqui nesse site vou abordar principalmente sobre o ggplot2 e sobre o dplyr.

R Studio

Para melhorar sua experiência com a linguagem R recomendamos a utilização do ambiente de desenvolvimento R Studio.
Conseguimos aproveitar melhor dos pacotes para R dentro dessa IDE, por possuir uma parte para download de packages e até visualização dos dados que você importar.
Primeiro você terá que instalar a linguagem R no seu computador, podemos encontrar aqui neste link o download pra Windows. E para fazermos download da IDE, R Studio, podemos encontrar neste link também para qualquer sistema operacional que você esteja usando.
